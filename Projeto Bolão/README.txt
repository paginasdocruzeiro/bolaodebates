Bolão do Cruzeiro Debates — projeto web funcional

Ficheiros:
- index.html
- styles.css
- app.js

Como usar:
1. Coloque o ficheiro da logo fornecida pelo utilizador com o nome exato Bolao1.png na mesma pasta destes ficheiros.
2. Abra o ficheiro index.html no navegador.
3. O sistema funciona localmente com localStorage, sem depender de backend externo.

Funcionalidades incluídas:
- Home, login, dashboard, ranking, rodada, histórico, estatísticas e admin
- Login com nome + PIN de 4 dígitos
- Participantes pré-cadastrados
- Admins: Ivo e Samuel
- Criação e edição de rodadas
- Encerramento manual ou automático
- Registo e edição de apostas
- Pontuação automática: exato 3, parcial 1, erro 0
- Ranking geral, ranking da rodada e ranking de consistência
- Histórico por jogador
- Estatísticas gerais
- Geração de mensagem para WhatsApp com encodeURIComponent
- Versão printável
- Favicon apontado para Bolao1.png
